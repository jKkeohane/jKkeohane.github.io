About Me

I built this website firstly to learn how to apply HTML, CSS and JavaScript in development, finding out how to do this using YouTube tutorials and examples from around the web. It also serves as a useful tool to illustrate some of the key studies I have undertaken during my university education at:

The University of Manchester where I am studying for an MSc in Geographical Information Sciences completing the main programme of courses in May 2023.

The University of Bristol where I studied for a BSc in Geography and graduated in 2021.

Between Bristol and Manchester I worked for Landmark Information group as a Spatial Data Technician working with a purpose-built GIS API to match addresses to spatial references for an OFGEM contracted project whilst also working as a climbing instructor.

GEO-PORTFOLIO

GEO-PORTFOLIO

Throughout my education I have been fortunate to cover a range of topics, from research and analysis covering global migration and development dynamics and sustainable neighbourhood planning to remote sensing imagery interpretation with multispectral optical and Synthetic Aperture Radar products and digital terrain model construction using UAV imagery and multi-view stereo methods.

Building confidence and proficiency with GIS applications including QGIS, ArcGIS Pro and ArcGIS Online has been crucial to accessing higher levels of analysis for multiple geographic variables, particularly when using programming outputs generated using the r and python programming languages as well as information obtained using JavaScript with Google Earth Engine and ESA Snap.

I am currently working on my dissertation exploring geographic natural language processing and looking forward to the first stages in my career as a geospatial scientist.
